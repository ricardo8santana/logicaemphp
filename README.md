
Este projeto apresenta conceitos fundamentais de lógica de programação aplicados em PHP. O objetivo é demonstrar como trabalhar com variáveis, operadores, estruturas condicionais, loops, funções e arrays de forma prática e clara. Todos os exemplos utilizam PHP puro, sem dependências externas, para manter o foco na lógica e no raciocínio. É necessário ter PHP instalado (versão 7.4 ou superior) e um servidor local como XAMPP ou Laragon para executar os scripts. Os arquivos foram criados para ilustrar situações comuns, como cálculos, validação de dados e controle de fluxo, servindo como base para iniciantes ou para revisão antes de avançar para frameworks. Para rodar os exemplos, basta clonar o repositório, abrir os arquivos e executar via terminal ou servidor local. O projeto não inclui estilização ou banco de dados, mantendo a simplicidade e a clareza do aprendizado.
