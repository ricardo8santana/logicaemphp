Lógica Direcionada em PHP
📌 Descrição
Este projeto demonstra como implementar uma lógica direcionada em PHP, utilizando estruturas condicionais para tomar decisões com base em regras definidas. Ideal para sistemas que precisam executar ações diferentes dependendo de parâmetros ou estados.

✅ Funcionalidades

Estrutura de decisão baseada em condições (if, switch).
Separação de lógica em funções para melhor organização.
Exemplo prático de direcionamento de fluxo (simulação de escolha de rotas ou ações).


🛠 Tecnologias

PHP (>= 8.2.12)
Servidor local (XAMPP, WAMP ou PHP embutido)
